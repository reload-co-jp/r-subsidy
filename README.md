# 中小企業向け補助金ポータルサイト R subsidy

## 1. 概要

### 1.1 サイトの目的
中小企業・個人事業主が、自社属性を先に入力することで、自社に合った補助金情報を絞り込んで閲覧できるポータルサイトを提供する。

本サイトは、単なる補助金一覧ではなく、以下を重視する。

- 自社向けの補助金を探しやすいこと
- 国の補助金に加えて東京都の補助金も探せること
- 難解な制度情報を平易に整理して表示すること
- 静的サイトとして低コスト・高速に運用できること

### 1.2 想定ユーザー
- 中小企業の経営者
- 小規模事業者
- 個人事業主
- 補助金を初めて探す担当者
- 東京都内の事業者、または東京都の制度にも関心がある事業者

### 1.3 提供価値
- 自社属性を入力すると、条件に合う補助金を優先表示できる
- 「全国対象」と「東京都対象」を同時に比較できる
- 補助金ごとの対象条件、用途、上限額、補助率、募集状況が一目でわかる
- JSONデータを更新するだけでサイト全体を更新できる

---

## 2. システム方針

### 2.1 基本方針
- DBは使わず、JSONファイルをデータソースとする
- サイトは静的生成する
- ユーザーの入力内容はブラウザ上で保持し、サーバー保存は行わない
- 検索・絞り込み・スコアリングはフロントエンド上で実行する

### 2.2 想定構成
- フロントエンド: Next.js
- データソース: JSONファイル
- 配信: GitHub Pages
